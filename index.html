Legends of Runeterra Card Gallery

React, Javascript, HTML & CSS

+

I created a responsive and user-centered card gallery for the popular card game Legends of Runeterra in React with filtering, sorting, and aggregating functionalities.


Legends of Runeterra is a popular online card game. This website aims to help players find cards suitible for their playstyles and help new players learn about the cards and their information. The user can filter the cards by their region and mana cost, sort the cards by mana cost, and add cards to a deck.


This website is built with expandability in mind. That is, new features and cards can be easily added in the future.


This website considers several usability and UIUX principles. It considers effectivity, efficiency, comprehensible design, learnability, and consistent design.


Visit the website!
